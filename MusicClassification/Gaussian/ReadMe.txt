To run the unll follow the steps
1. Go into unll_niave.py and check update file & path variables for...
a. the labels file
b. training data set folder
c. test data set folder
2. Once proper values are entered for file and path variables simply execute program
3. File with the labeled test songs will appear as ngauss_tstlabels.csv

can find song data at
https://www.kaggle.com/c/music-genre-classification-audio
