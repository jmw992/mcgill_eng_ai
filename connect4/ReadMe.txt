Connect 4 adverserial search

To run the program 

Go into main.cpp and select the desired board to test
Set the desired time or depth limit
Select the desired search method / minimax / alpha beta 
Complie the program with
	g++ main.cpp -o main1
Execute the program
	./main1

if the c++ compiler is not installed on the linux machine install it with
	sudo apt-get install g++
